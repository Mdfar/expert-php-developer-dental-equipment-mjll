Dental E-commerce Engine by Staqlt
Features

Dynamic Catalog: Specialized attributes for dental equipment (voltage, warranty, sterilization grade).

Secure B2B Portal: Role-based access for clinics and purchasing managers.

Real-time Synchronization: Stock levels sync across the frontend and admin panel instantly.

Payment Security: Fully integrated Stripe checkout for high-ticket medical transactions.

Setup

Upload /includes and /admin to your PHP 8.2+ server.

Import database.sql into MySQL.

Configure config.php with your DB and Stripe API credentials.
